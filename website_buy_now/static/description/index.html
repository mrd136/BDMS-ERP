Website Buy Now

Faster checkouts and higher conversions with "Buy Now"

Odoo Website Buy Now Module adds a "Buy Now" icon on each product on the shop page and product page. Hence, the module makes the checkout process quicker which reduces cart abandonment.

Information

User Guide

Requirements

  • The module adds Buy Now button to the products on Shop Page and product page.

  • The "Buy Now" button will now appear next to products as an icon.

  • The buyers can just click on the "Buy Now" button which would take them directly to the shipping page while skipping the cart and process quick checkout in as small as 2 steps.

Key Features

  • The module facilitates quicker checkout for customers looking to buy a one product at a time.

  • It adds a Buy Now button individually to the products on both the shop page and product page on Odoo website.

  • The customers can just click on the "Buy Now" button and go to shipping information page, skipping the cart altogether and proceed with the checkout.

  • When Buy Now is clicked, the products present in the cart (if any) gets automatically hidden and the cart gets reset to show just the clicked product. Once the order is placed, the cart gets automatically recovered with the original products in the list.

  • If Buy Now button is clicked on multiple browsers or multiple tabs from the same account, the system displays an error while making payments to prevent conflicting orders from same Buy Now cart.
