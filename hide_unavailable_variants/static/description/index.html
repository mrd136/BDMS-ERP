Hide/Cancel-Out Unavailable Variants

Hide/Cancel-Out the unavailable variants of the product in website shop

Few Words About The Hide/Cancel-Out Unavailable Variants:

When some of the variants for a product are not available, This app will give you option to "Cancel-Out" or "Hide" those variants in your website shop.

Work Flow: How The Hide/Cancel-Out Unavailable Variants Works
Set view Type:
  • Go to the Attributes and create or select the attribute and open it
  • Set the Unavailable Variant View Type to "Cancel Out" or "Hide" or "None" as your requirement
Product Website View:
  • When "Cancel Out" option is selected
  • Unavailable variants are cancelled, user can not select them.
Product Website View:
  • When "Hide" option is selected
  • Unavailable variants are hidden, user can not see them.
